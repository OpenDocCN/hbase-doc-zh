## 下载

### Docker

```
docker pull apachecn0/hbase-doc-zh
docker run -tid -p <port>:80 apachecn0/hbase-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g hbase-doc-zh
hbase-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
